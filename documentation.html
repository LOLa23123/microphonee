Installation Guide

Follow these steps to install Equalizer APO on your Windows system.

System Requirements

  • Windows 7 or later
  • Audio device that supports APO
  • Administrator privileges

Installation Steps

  1. Download the latest version from our website
  2. Run the installer as administrator
  3. Select your audio device during installation
  4. Restart your computer if prompted
Note: Make sure to select the correct audio device during installation. You can always change this later through the configuration panel.

Troubleshooting

Common issues and their solutions.

Common Problems

No Sound After Installation

Check if your audio device is properly selected in the configuration panel.

High CPU Usage

Try reducing the number of active filters or using simpler filter types.
